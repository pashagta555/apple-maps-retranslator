# apple-maps-retranslator

Developer link: [t.me/pashagta](https://t.me/pashagta)

This project is intended for people who do not want to use defective Yandex maps, but at the same time their work forces them to use them to one degree or another to simplify the work (for example, Yandex couriers, whose choice of opening links to an address in external navigation applications is limited to defective Yandex maps). This application has the ability to open links intended for Yandex maps and redirect them to apple maps (you must not have the yandex maps application installed for this function to work). You can also paste the link into the application itself and open the link in apple maps.
